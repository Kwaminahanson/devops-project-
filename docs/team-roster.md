## Roles & Responsibilities

### DevOps Engineer
- Maintains CI/CD pipeline configuration
- Manages Docker infrastructure
- Owns the docs-as-code platform
- Monitors deployment environments

### Backend Developer
- Writes and maintains application code
- Reviews infrastructure changes affecting the app
- Documents API changes in this repo

### QA Engineer
- Owns test plans for each environment
- Signs off on UAT before production deployments
- Documents known issues and workarounds

### Release Manager
- Approves production deployments
- Manages change control process
- Escalation point for deployment failures
